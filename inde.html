Componentes de la Computadora

CPU Memoria RAM Tarjeta Gráfica Almacenamiento

CPU (Unidad Central de Procesamiento)

El CPU es el cerebro de la computadora. Se encarga de procesar las instrucciones y ejecutar las tareas necesarias para el funcionamiento del sistema.

Memoria RAM

La memoria RAM almacena temporalmente los datos y programas que se están utilizando, permitiendo un acceso rápido para el procesador.

Tarjeta Gráfica (GPU)

La GPU se encarga de procesar los gráficos y las imágenes, siendo esencial para videojuegos, diseño y aplicaciones de inteligencia artificial.

Almacenamiento

El almacenamiento guarda permanentemente los archivos y programas, incluso cuando la computadora está apagada. Puede ser un disco duro (HDD) o una unidad de estado sólido (SSD).
